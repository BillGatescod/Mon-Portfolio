# Mon-Portfolio

Portfolio personnel avec une esthétique terminal / développeur, construit en HTML/CSS/JS.

 **Live :**https://github.com/BillGatescod/Mon-Portfolio

## Aperçu

Ce site présente mon parcours, mes projets et mes compétences en tant qu'étudiant en Génie Logiciel (cycle ingénieur) à l'IAI Cameroun. Le design s'inspire d'un terminal de développeur, avec une palette sombre et des accents colorés.

## Design

- **Thème :** dark terminal / développeur
- **Palette :** navy, orange, purple, cyan
- **Typographies :** JetBrains Mono (code/texte technique), Syne (titres)
- **Responsive :** adapté mobile, tablette et desktop

## Structure du projet

```
mon-portfolio/
├── index.html           # Page d'accueil
├── style.css            # Styles principaux
├── script.js            # Interactions et animations
├── assets/
│   ├── CV.pdf           # Curriculum Vitae
│   └── projects/        # Images des projets
└── README.md
```

## Sections

- **Accueil** — présentation et pitch personnel
- **À propos** — parcours académique et compétences
- **Projets** — réalisations techniques (web, réseau, mobile, etc.)
- **Contact** — moyens de me joindre

## Technologies utilisées

- HTML5 / CSS3
- JavaScript (vanilla)
- Google Fonts (JetBrains Mono, Syne)

## Déploiement

Le site est hébergé via **GitHub Pages**.  
Toute modification poussée sur la branche `main` est automatiquement déployée en quelques minutes.

```bash
git add .
git commit -m "Mise à jour du site"
git push origin main
```
